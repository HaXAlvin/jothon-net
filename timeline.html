span 完成報名後，隨時都可以進行提案登記。

11/27
活動開場/ Opening
09:00
報到/ Check In

準備好參加活動了嗎？報到的流程如下： 1. 出示你的報到碼 2. 掛好自己的名牌 3. 貼好技能貼紙 4. 登入 WIFI 網路 5. 找個好位置

10:00
三分鐘提案/ Project Pitch

依照提案登記的順序輪流簡報，每組三分鐘。簡報重點： 想要做什麼 為什麼要做 需要誰幫忙 是否開源

10:40
自我介紹/ Self Introduction

首先全體參與者一起說「大家好」，接著逐一的向大家介紹自己的名字與三個關鍵字。

11:00
分組開工/ Group Hacking

提案與自介完成後，提案人領取桌牌與臂章裝備以識別專案，參與者各自尋找感興趣的專案分組帶開囉！ 分組開幹 三餐 ( 早午餐 / 午餐 / 下午茶 ) * 跨界交流

15:00
短講/ Short Talks

一場八分鐘的短講，開放登記，題目隨喜

17:00 前
登記成果報告/ Sign up for Project Demo
17:00
成果報告/ Project Demo

一組五分鐘的成果報告，無論進度如何都可以談談今天的進展喔！
